# Project Context

## Product

PM Studio is an AI-native product manager workspace. It turns an initial idea or PRD into structured planning artifacts and previewable prototypes.

## Target Users

- Product managers preparing early product proposals.
- Founders validating startup ideas.
- Operators and independent builders who need a fast product pack.
- Internal teams preparing project approval materials.

## Target Outputs

For Idea-to-Product Pack:

- Product positioning.
- Target users.
- User pain points.
- Value proposition.
- PRD draft.
- Core feature list.
- User flow.
- Prototype page structure.
- Market opportunity.
- Competitor analysis direction.
- MVP scope.
- Roadmap.
- Executive summary.

For PRD-to-Prototype:

- Extracted core features.
- User task flow.
- Information architecture.
- Prototype description.
- Previewable page prototype.

## Product Tone

Clear, practical, operator-focused, and artifact-first. The UI should feel like a working studio, not a marketing landing page generator.
