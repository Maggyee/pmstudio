# PM Studio Website Prototype

A Next.js App Router prototype for PM Studio. It uses mock data only and does not connect to a database, authentication, or real AI APIs.

## Run locally

```bash
npm install
npm run dev
```

Open:

- `http://localhost:3000/`
- `http://localhost:3000/app`
- `http://localhost:3000/app/projects/demo`

## Scripts

```bash
npm run lint
npm run build
```

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Handwritten shadcn/ui-style components
- Mock data in `lib/mock-data.ts`
